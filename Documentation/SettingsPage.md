# Settings Page

The Settings Page allows you to configure BlendHub preferences, customize application behavior, and manage various aspects of your experience. Access all configuration options in one organized location.

### Header Section
At the top of the page, you'll find the main title:

- **Settings Title** - Clear page header indicating configuration options

### Main Content Area
The page contains multiple configuration sections organized in expandable cards:

## Appearance & Behavior

### App Theme
Choose the visual theme for BlendHub:

- **Theme ComboBox** - Select from available themes:
  - **Auto** - System theme preference
  - **Light** - Light theme interface
  - **Dark** - Dark theme interface
- **Theme Description** - "Choose which theme you want to app to use"
- **Theme Icon** - Visual indicator for theme selection

### Username Configuration
Set your personal name for greetings:

- **Username TextBox** - Enter your preferred name
- **Description** - "Set name that will be used for greetings throughout app"
- **User Icon** - Visual indicator for user settings
- **Auto-Save** - Changes are saved automatically when focus is lost

### Default Page Selection
Choose which page opens when BlendHub starts:

- **Page ComboBox** - Select from available options:
  - **Home** - Main dashboard page
  - **Download** - Blender version management
  - **Backup** - Backup creation and management
  - **Restore** - Data restoration from backups
  - **Sync** - Cross-version synchronization
  - **Projects** - Project management interface
- **Page Description** - "Choose which page should open when you start BlendHub"
- **Home Icon** - Visual indicator for page selection

## General Settings

### Custom Blender Installations
Add custom Blender installations to show on Home page:

- **Custom Blenders Section** - Expandable list of custom installations
- **Add Blender Button** - Browse and add Blender executables
- **Installation List** - Display of all custom Blender paths
- **Path Display** - Shows full path to each Blender executable
- **Change Path Button** - Modify existing Blender installation path
- **Remove Button** - Delete custom Blender installation from list

## Backup & Restore Settings

### Backup Location Configuration
Set where your backup files are stored:

- **Backup Location Card** - Configure default backup storage location
- **Path Display** - Shows current backup folder path
- **Browse Button** - Open folder picker to choose backup location
- **Path Text** - Displays full backup folder path with ellipsis for long paths

## Default Project Folders

### Project Folder Structure
Configure default folders created for new projects:

- **Default Folders Section** - Expandable list of project folder templates
- **Folder Name Input** - Configure names for default project folders
- **Add Folder Button** - Create new default folder template
- **Folder List** - Display of all configured default folders
- **Remove Button** - Delete unwanted folder templates

## File Type Associations

### Default File Launchers
Set default programs for opening different file types:

- **File Launchers Section** - Expandable list of file type associations
- **Extension Input** - Configure file extension (e.g., .psd)
- **Program Path Input** - Set path to associated application
- **Browse Button** - Choose executable for file association
- **Program Path Display** - Shows full path to associated program
- **Add Launcher** - Create new file type association
- **Remove Button** - Delete existing file association

## How to Use

### Customizing Appearance
1. **Choose Theme** - Select Auto, Light, or Dark theme
2. **Set Username** - Enter your name for personalized greetings
3. **Select Default Page** - Choose which page opens on startup
4. **Apply Changes** - Settings are saved automatically

### Managing Blender Installations
1. **Add Custom Blender** - Click "Add Blender Executable" button
2. **Browse to Blender** - Select Blender installation folder
3. **Configure Path** - Set custom installation path
4. **Manage List** - Add, edit, or remove custom installations

### Configuring Backups
1. **Set Backup Location** - Choose where to store backup files
2. **Browse Folder** - Use folder picker to select backup location
3. **Verify Path** - Ensure backup location is accessible
4. **Apply Changes** - Settings are saved automatically

### Setting Up Project Folders
1. **Configure Templates** - Set up default folder structures
2. **Add Folders** - Create custom folder templates
3. **Name Folders** - Use descriptive names for organization
4. **Manage Templates** - Edit or remove folder configurations

### File Association Setup
1. **Add File Type** - Configure new file type association
2. **Set Extension** - Enter file extension (e.g., .psd, .ai)
3. **Choose Program** - Browse to associated application
4. **Configure Path** - Set full path to executable
5. **Test Association** - Verify file opens with correct program

## Troubleshooting

### Common Issues
- **Theme Not Applying** - Restart application to apply theme changes
- **Username Not Saving** - Ensure you press Enter or click away from field
- **Default Page Not Working** - Verify page selection is valid
- **File Associations Not Working** - Check program paths and permissions

### Solutions
- **Restart Application** - Some settings require application restart
- **Check Permissions** - Ensure application has necessary system access
- **Verify Paths** - Confirm file and program paths are correct
- **Reset Settings** - Use reset option if settings become corrupted
