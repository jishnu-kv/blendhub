# Home Page

The Home Page is your main dashboard in BlendHub, providing quick access to all features and showing your installed Blender versions and recent projects.

![Home Page](../src/Assets/screenshots/home.png)

## Page Layout

### Welcome Message
At the top of the page, you'll see a personalized greeting message that welcomes you back to BlendHub.

### Navigation Cards
Below the welcome message, you'll find horizontal scrollable cards that provide quick access to the main features:

- **Download Blender** - Download different Blender versions
- **Create Backup** - Save your Blender preferences securely
- **Restore Data** - Recover configuration from previous backups  
- **Sync Versions** - Synchronize settings across Blender versions

Each card has an icon, title, and description. Click any card to navigate to that feature.

### Installed Blender Versions Section
This section displays all Blender versions that BlendHub has automatically detected on your computer.

**Automatic Scanning**: BlendHub automatically scans common installation folders including:
- Program Files\Blender Foundation
- Steam installation directory
- WindowsApps (Microsoft Store installations)
- Local application data folders

Each detected version shows:
- **Version Name** (e.g., "Blender 4.2.0")
- **Installation Path** (full path to the executable)
- **Open Blender Button** - Launch that specific Blender version directly
- **Config Folder Button** - Click the arrow to open the configuration folder for that Blender version

**Refresh Button**: Click the refresh icon (circular arrow) to re-scan your system for newly installed Blender versions.

### Recent Projects Section
This section shows your 5 most recently created or accessed Blender projects.

Each project is displayed as a card with key project information. Click on any project card to open it directly.

**View All Projects Button**: Click the arrow icon to see all your projects in the Project page.

**Empty State**: If you don't have any projects yet, you'll see a message encouraging you to create your first project.

## How to Use

### Getting Around
1. **Navigate to Features**: Click any navigation card to access that feature
2. **Launch Blender**: Click "Open Blender" next to any detected version to start it
3. **Open Recent Projects**: Click on any recent project card to open it
4. **Refresh Versions**: Click the refresh button if you've installed new Blender versions

### For First-Time Users
1. **Explore Navigation Cards**: Click each card to discover what BlendHub can do
2. **Check Detected Versions**: See which Blender versions BlendHub found on your system
3. **Create Your First Project**: Use the Projects card to get started

## Tips for Beginners

- **Start Here**: The Home Page is the best place to begin exploring BlendHub
- **Quick Launch**: Use the "Open Blender" buttons to start Blender without navigating through folders
- **Automatic Detection**: BlendHub finds your Blender installations automatically - no setup needed
- **Recent Access**: Your most recent projects appear here for easy access
- **Refresh After Installation**: Click refresh after installing new Blender versions so BlendHub can detect them

## Understanding the Interface

### Navigation Cards
- **Horizontal Scrolling**: Swipe or scroll horizontally to see all cards
- **Visual Icons**: Each card has a unique icon for easy identification
- **Clear Descriptions**: Each card explains what that feature does

### Version Detection
- **No Configuration Needed**: BlendHub automatically finds your Blender installations
- **Multiple Sources**: Detects installations from Program Files, Steam, and Microsoft Store
- **Version Specific**: Shows exact version numbers for each detected installation

### Project Management
- **Recent First**: Your most recently used projects appear first
- **Quick Access**: Click any project to open it immediately
- **Visual Preview**: Project cards show key information at a glance
