# Project Detail Page

The Project Detail Page provides comprehensive management for individual projects, including file browsing, note-taking, task management, and project statistics. Access all project information and tools in one organized interface.

![Project Detail Page](../src/Assets/screenshots/project.png)

## Page Layout

### Header Section
At the top of the page, you'll find project information and main actions:

- **Back Button** - Navigate back to the project list
- **Project Title** - Large text showing project name
- **Project Path** - Full file system path to project folder
- **Edit Button** - Modify project settings and properties
- **Delete Button** - Remove project from BlendHub
- **Open Folder Button** - Open project location in file explorer
- **Open Project Button** - Launch project in Blender with version selection

### Quick Stats Section
Below the header, you'll find key project information in card format:

- **Blender Version** - Associated Blender version for this project
- **Blend File** - Main .blend file for the project
- **Created** - Date when project was created
- **Last Modified** - Date of last project modification
- **Project Size** - Total size of project folder

## How to Use

### Project Navigation
1. **Open Project** - Click on project from main project list
2. **View Details** - See comprehensive project information
3. **Navigate Tabs** - Switch between Locations, Notes & Tasks, and Files
4. **Access Actions** - Use header buttons for project operations

### Managing Project Locations
1. **Browse Folders** - Click on folder cards to navigate
2. **View Contents** - See files and subfolders in each location
3. **Refresh View** - Click Refresh to scan for new content
4. **Navigate Back** - Use browser navigation to return to parent folders

### Working with Notes
1. **Add Note** - Click "Add Item" → "Add Note"
2. **Enter Content** - Type your note content
3. **Save Note** - Note is automatically saved
4. **Edit Notes** - Click edit button to modify existing notes
5. **Delete Notes** - Remove unwanted notes with delete button

### Managing Tasks
1. **Add Task** - Click "Add Item" → "Add Task"
2. **Set Priority** - Choose priority level for task
3. **Track Progress** - Check off completed tasks
4. **Edit Tasks** - Modify task content or priority
5. **Delete Tasks** - Remove completed or unnecessary tasks

### File Management
1. **View Files** - Browse all project files in Files tab
2. **Add Files** - Click "Add File" to import new files
3. **File Operations** - Use file actions for management
4. **File Information** - View file details and properties

## Troubleshooting

### Common Issues
- **Project Not Found** - Warning bar shows if project folder is missing
- **Files Not Loading** - Check if project folder exists and is accessible
- **Tasks Not Saving** - Ensure project folder has write permissions
- **Navigation Issues** - Use refresh button to update folder view

### Solutions
- **Check Path** - Verify project folder exists at specified location
- **Permissions** - Ensure BlendHub has access to project folder
- **Refresh Content** - Use refresh buttons to update views
- **Restart Application** - Some issues may require application restart
